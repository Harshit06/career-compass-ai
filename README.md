# 📖 Overview

**Career Compass AI** is an AI-powered web application designed to help users make informed career decisions. By analyzing skills, interests, and career goals, the platform generates personalized career recommendations and learning roadmaps using modern AI technologies.

Whether you're a student choosing a career path, a fresher preparing for placements, or a professional planning a career switch, Career Compass AI serves as your intelligent career advisor.

# 🚀 Installation

## 1. Clone Repository

```bash
git clone https://github.com/Harshit06/career-compass-ai.git
```

---

## 2. Move into Project Directory

```bash
cd career-compass-ai
```

---

## 3. Create Virtual Environment

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / Mac

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 5. Configure Environment Variables

Create a `.env` file and add your Gemini API key.

```env
GEMINI_API_KEY=YOUR_API_KEY
```

---

## 6. Run the Application

```bash
streamlit run app.py
```
